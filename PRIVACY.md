# Политика конфиденциальности приложения HomeBudget

**Последнее обновление:** 5 апреля 2026 г.

## 1. Общие положения

Приложение **HomeBudget** (далее — «Приложение») разработано для ведения учёта личных финансов. Настоящая политика описывает, какие данные собираются и как они используются.

## 2. Какие данные собирает приложение?

**Приложение НЕ собирает и НЕ передаёт ваши личные данные на внешние серверы.**

Вся информация, которую вы вводите (транзакции, категории, настройки пользователя), хранится **исключительно локально на вашем устройстве** в базе данных SQLite. Разработчик не имеет доступа к этим данным.

## 3. Какие разрешения запрашивает приложение?

| Разрешение | Зачем нужно |
|------------|-------------|
| **Уведомления (POST_NOTIFICATIONS)** | Для отправки периодических финансовых отчётов (ежедневно, еженедельно, ежемесячно). Вы можете отключить уведомления в настройках приложения или системы в любой момент. |
| **Доступ к файлам (READ/WRITE_EXTERNAL_STORAGE / MANAGE_EXTERNAL_STORAGE для Android 11+)** | Для сохранения финансовых отчётов в формате PDF на ваше устройство. Файлы создаются только по вашему запросу (кнопка «Экспорт в PDF»). |

## 4. ИИ-ассистент (YandexGPT)

Функция ИИ-ассистента работает **только при вашем активном желании**:

* Вы **самостоятельно** вводите API-ключ и Folder ID от Yandex Cloud в настройках приложения.
* Без вашего ключа ИИ-ассистент не работает и не отправляет никаких данных.
* Когда вы задаёте вопрос в чате, приложение отправляет в API YandexGPT:
  * Текст вашего вопроса
  * Данные о транзакциях за выбранный период (сумма, тип, категория, дата, заметка)
* Все запросы идут по защищённому протоколу **HTTPS**.
* Yandex Cloud обрабатывает данные в соответствии со своей политикой конфиденциальности.

## 5. Безопасность данных

* Пароли пользователей хранятся в **хешированном виде** (SHA-256).
* Сессия пользователя сохраняется в зашифрованном хранилище (`EncryptedSharedPreferences`).
* API-ключ для YandexGPT хранится в зашифрованном виде (Android Keystore).
* SQL-запросы используют параметризацию для защиты от SQL-инъекций.

## 6. Передача данных третьим лицам

Приложение **не передаёт** ваши данные третьим лицам, за исключением:
* Запросов к YandexGPT API (только при использовании ИИ-ассистента, только с вашего согласия).

## 7. Права пользователя

Вы можете в любой момент:
* Удалить все свои данные, очистив данные приложения в настройках Android или удалив приложение.
* Отключить уведомления в настройках приложения.
* Не использовать ИИ-ассистент (просто не вводите API-ключ).
