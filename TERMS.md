## 2. Регистрация и учётная запись

2.1. Для использования всех функций Приложения необходима регистрация.

2.2. При регистрации Пользователь указывает:
   - логин (уникальное имя)
   - email (не верифицируется, может быть любым)
   - пароль (не менее 8 символов, буквы и цифры)
   - секретный вопрос
   - ответ на секретный вопрос

2.3. Пользователь самостоятельно выбирает и запоминает свои учётные данные. Разработчик не имеет доступа к паролю (хранится в хешированном виде).

2.4. Пользователь несёт полную ответственность за сохранность своих учётных данных (логин и пароль).

2.5. Разработчик не несёт ответственности за несанкционированный доступ к учётной записи Пользователя, если это произошло не по вине Разработчика.

2.6. При бездействии Пользователя в течение 30 минут Приложение автоматически запрашивает повторный ввод пароля для обеспечения безопасности.

## 3. Финансовые данные и конфиденциальность

3.1. Все финансовые данные, вводимые Пользователем (транзакции, категории, заметки), хранятся **исключительно локально** на устройстве Пользователя в базе данных SQLite.

3.2. Разработчик **не имеет доступа** к финансовым данным Пользователя и **не собирает** их на внешних серверах.

3.3. Пользователь самостоятельно отвечает за сохранность своих данных. Рекомендуется регулярно делать резервные копии (экспорт данных в PDF).

3.4. При удалении Приложения все данные Пользователя удаляются безвозвратно, если ранее не был сделан экспорт.

## 4. ИИ-ассистент (YandexGPT)

4.1. ИИ-ассистент является дополнительной функцией и работает **только при активном желании Пользователя**.

4.2. Для работы ИИ-ассистента Пользователь **самостоятельно** получает API-ключ и Folder ID в сервисе Yandex Cloud и вводит их в настройках Приложения.

4.3. Без введённого API-ключа ИИ-ассистент не работает и не отправляет никаких данных.

4.4. При использовании ИИ-ассистента Приложение отправляет в API YandexGPT:
   - Текст вопроса Пользователя
   - Данные о транзакциях за выбранный период (сумма, тип, категория, дата, заметка)

4.5. Все запросы к YandexGPT передаются по защищённому протоколу HTTPS.

4.6. Разработчик **не несёт ответственности** за ответы, сгенерированные ИИ-ассистентом. ИИ может давать ошибочные или неточные рекомендации.

4.7. Пользователь использует ИИ-ассистента на свой страх и риск.

## 5. Разрешения приложения

5.1. Приложение запрашивает следующие разрешения:

| Разрешение | Цель использования |
|------------|-------------------|
| **Уведомления (POST_NOTIFICATIONS)** | Для отправки периодических финансовых отчётов (ежедневно, еженедельно, ежемесячно). Пользователь может отключить уведомления в настройках Приложения или системы. |
| **Доступ к файлам (READ/WRITE_EXTERNAL_STORAGE / MANAGE_EXTERNAL_STORAGE)** | Для сохранения финансовых отчётов в формате PDF на устройство Пользователя. Файлы создаются только по явному запросу Пользователя (кнопка «Экспорт в PDF»). |

5.2. Приложение не запрашивает и не требует других разрешений (геолокация, контакты, камера, микрофон, SMS).
